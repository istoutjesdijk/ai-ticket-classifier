Dit is een plugin voor osticket die met behulp van de openai of anthropic api de ticket een prioriteit en categrorie/onderwerp geeft.
Dit moet gebeuren als een ticket wordt aangemaakt of als er een nieuw bericht van de klant aan een ticket wordt toegevoegd. Steeds op basis van het laatste bericht.

Ter referentie kan je de codebase van osticket en een voorbeeld plugin vinden in:
osticket/
ai-response-generator/

Technische eisen
code moet simpel, modern en makkelijk te onderhouden zijn
goed gedocumenteerde code
Gebruik relevante skills of plugins
Echt een plugin, dus zoveel mogelijk de interne functies / libaries / frameworks van osticket gebruiken
Klaar voor productie

Laat de gebruiker kiezen tussen belangrijke keuzes als deze gemaakt moeten worden.

Omgeving:
Je hebt toegang tot een volledig Ubuntu ontwikkelomgeving met oa PHP, MariaDB, Apache, nginx etc
